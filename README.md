# AppDemo
app开发框架，一个新闻阅读App框架，Material Design + Retrofit + Glide
# 模块
- 首页：新闻头条（聚合数据API）
- 资讯：一写Android，ios，前段等资料（代码家的gank.io API）
- 美女：妹纸（代码家的gank.io API福利）
- 我的：一些小功能
